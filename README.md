# Algoritmos Recursivos em C
Coleção de exercícios práticos desenvolvidos para aprofundar o domínio sobre recursividade em C. O repositório contém a implementação de algoritmos clássicos de matemática e manipulação de estruturas, abordando desde operações básicas até funções complexas. O projeto demonstra a aplicação de casos base, chamadas recursivas e a gestão de pilha de execução, sendo uma excelente vitrine para o entendimento de lógica de programação avançada e eficiência algorítmica.
## Tecnologias Utilizadas
- **C**
- **Manipulação de Strings**
- **Algoritmos Matemáticos**: Cálculos de MDC (Máximo Divisor Comum) e a complexa função de Ackermann.
